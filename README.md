## Description

Domain-Driven Design + Clean Architecture for node.js

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests + e2e tests
$ npm run test

# unit tests
$ npm run test:unit

# e2e tests
$ npm run test:e2e

# test coverage (unit tests)
$ npm run test:unit:cov

# test coverage (e2e tests)
$ npm run test:e2e:cov
```
